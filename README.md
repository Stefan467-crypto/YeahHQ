## Настройка после деплоя на Railway

В `config.py` одна строка требует вашего внимания:

```python
MINI_APP_URL = "https://your-project.up.railway.app/miniapp"
```

Замените `your-project` на реальное название вашего Railway домена.
Домен виден в Railway → ваш проект → Settings → Domains.

## Структура файлов

```
bot.py          — основной код бота
config.py       — токен, настройки
database.py     — база данных SQLite
miniapp.html    — мини-приложение (раздаётся автоматически на порту 8080)
requirements.txt
start.sh
```

## Как работает мини-приложение

- При запуске бота автоматически стартует веб-сервер на порту **8080**
- `miniapp.html` доступен по адресу: `https://ваш-домен.up.railway.app/miniapp`
- После того как узнаете домен — замените `MINI_APP_URL` в config.py и задеплойте снова
