## Что нового в v3.0

### ✅ Исправлено
- Юзернейм везде заменён на `@YeahHQ_Bot`
- `@Stefanwt` теперь отображается как **Владелец** (не совладелец)
- Исправлен баг с платными функциями — теперь при недоступной функции показывается **кнопка «Купить»**

### 🆕 Новое

**Меню команд с кнопками по разделам:**
- Нажимаешь «📋 Команды» → список разделов (Модерация, Должности, Развлечения, Настройки, Профиль)
- В разделе — список кнопок с командами
- Платные команды помечены `(платно)` в названии кнопки
- Внизу каждого платного раздела — кнопка «🛒 Купить платные функции»

**Добавление в группу кнопкой:**
- Нажимаешь «➕ Добавить в группу» → появляется **Inline-кнопка** (не просто текст)
- Бот автоматически запрашивает права администратора при добавлении
- Кнопка «❓ Зачем нужны права?» с пояснением

**Расширенная статистика `/botstats`:**
- Сколько пользователей (всего + за день/неделю/месяц)
- Сколько групп
- Сколько действий (всего + по периодам)
- **Заработано Stars ⭐** — за день, неделю, месяц, год, всё время
- Количество браков

**Новые команды:**
- `/note add|get|del|list` — заметки в группе
- `/filter keyword ответ` — автоответы по ключевым словам
- `/pin` / `/unpin` — закрепление сообщений
- `/chatstats` — статистика чата (доступна всем)
- `/allgroups` — список всех групп (владелец)

**Мини-приложение** (`miniapp.html`):
- 5 вкладок: Статистика, Функции, Топ, Команды, О боте
- График заработка по периодам
- Список функций с кнопками «Купить»
- Кнопка «Добавить в группу»
- Период статистики: день / неделя / месяц / год / всё время

**Приветствие** — длинное, без списка команд (команды теперь в кнопочном меню)

## Установка

```bash
pip install -r requirements.txt
python bot.py
```

## Конфиг (`config.py`)

```python
BOT_TOKEN = "..."         # Токен от @BotFather
BOT_USERNAME = "YeahHQ_Bot"
BOT_OWNER_USERNAME = "Stefanwt"
BOT_OWNER_INITIAL_IDS = [YOUR_ID]  # Ваш Telegram ID
MINI_APP_URL = "https://your-domain.com/miniapp"  # URL мини-приложения
```

## Мини-приложение

Задеплойте `miniapp.html` на любой хостинг (GitHub Pages, Vercel, Netlify).
Укажите URL в `MINI_APP_URL` в `config.py`.
Затем зарегистрируйте Mini App в @BotFather: Bot Settings → Menu Button → URL.
